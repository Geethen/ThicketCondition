Thicket Ecological Condition · Random Forest

Detecting intact thicket: threshold selection & band combination

The per-EFG Random Forest outputs a probability that each thicket pixel is intact. Part 1 sweeps a threshold τ on P(intact) to find where classification quality peaks and how the mapped intact area responds. Part 2 asks which threshold-selection method to use, and whether combining all three probability bands — via hand-crafted rules and symbolic regression — beats P(intact) alone. Part 3 repeats the sweep one-vs-rest for the moderate and severe classes too. Part 4 checks whether the spatial CV block is large enough using an empirical variogram. All accuracy is spatially cross-validated (out-of-fold).

Part 2

Which method, and can the three bands do better?

Two follow-up questions: (1) which threshold-selection rule from the remote-sensing literature should choose τ, and (2) whether combining p_intact, p_moderate and p_severe beats thresholding p_intact alone. Everything here is scored by held-out intact-class F1 on an independent set of spatial blocks.

Reference data

The training labels span the condition gradient plus two excluded classes. Only intact / moderate / severe are modelled; transformed is handled by the land-cover mask and bontveld is dropped.

Threshold-selection methods compared

Operating points chosen by the common remote-sensing criteria on the p_intact score. Liu et al. (2005) and Freeman & Moisen (2008) recommend max-Kappa or Youden's J / max-TSS; max overall accuracy is prevalence-biased.

Symbolic-regression Pareto front

Each point is a candidate formula in (p_i, p_m, p_s) found by PySR, plotted by complexity against held-out F1. Training error keeps falling with complexity, but held-out F1 does not improve — the extra terms fit noise, not signal.

Part 3

Ideal thresholds for all three condition classes

The same threshold sweep applied one-vs-rest to each condition class. For class C the positive label is C vs the rest and the score is that class's probability band (p_intact, p_moderate or p_severe). Accuracy is the same spatial out-of-fold CV; area is each class's full-model probability surface. Intact reproduces Part 1 exactly as a check.

Per-class ideal thresholds

Operating point at max Youden's J for each class, with ROC AUC (class separability) and the mapped area that qualifies at that threshold. Moderate is the hard middle class — lower AUC and a much flatter optimum.

Youden's J versus threshold, by class

Each curve is one class's J across τ; the dashed marker on each is its own τ*. A sharp peak means a well-defined threshold; a flat plateau (moderate) means the exact value matters little.

Mapped area versus threshold, by class

Area assigned to each class as its own τ tightens. These are independent one-vs-rest surfaces, so at low τ they overlap and need not sum to the total mappable area (19,106 km²).

Part 4

Is the 0.2° spatial CV block large enough?

Block cross-validation should use blocks at least as large as the range of spatial autocorrelation. The decisive check is the out-of-fold residual variogram: structure left after the Random Forest has fit the 64 Google embedding bands.

Empirical variogram and fitted range

Short-lag semivariance at the 2,083 training points, using geodesic distances and 2 km bins. Dots are empirical bins; lines are fitted exponential variograms. The vertical marker is the 0.2° CV block width, about 18.6 km east-west at the AOI latitude.
