Intact: dense, continuous canopy of woody thicket.
Moderate: patchy / thinned canopy, visible bare ground between clumps, browse lines.
Severe: mostly bare/eroded, sparse remnant bush, ephemeral herbaceous.
Use Not thicket for cultivation, plantation, urban, water, or clearly non-thicket vegetation.

Decision tree: thicket present? If no → Not thicket. If yes, canopy continuous → Intact; visibly opened but substantial → Moderate; sparse remnants/bare or eroded ground → Severe. If imagery cannot support a decision → Unsure + reason.

Shadows should not be read as canopy gaps; compare dates. Recent fire can temporarily expose ground—flag it. Regular field geometry indicates cultivation. For mixed pixels, judge the point centre and add the “mixed pixel” tag.
